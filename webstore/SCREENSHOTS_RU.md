# Скриншоты для Chrome Web Store (RU)

Рекомендуется 3–5 скриншотов (1280x800 или 640x400). Покажите реальную работу расширения.

1) **Главная панель расширения (popup)**
   - Открыто popup с кнопками СмартСкролл и VK Музыка.

2) **СмартСкролл в действии**
   - Плавающая панель на странице (кнопки Вверх/Вниз/Выбор/Стоп/Закрыть).
   - Можно показать подсветку контейнера при выборе.

3) **VK Музыка: массовые операции**
   - Окно редактирования плейлиста, видна плавающая панель с «Добавить/Снять/Стоп».
   - В строке статуса отображается прогресс.

4) **Экспорт в TXT**
   - Страница VK Музыки + момент запуска экспорта (например, скачивание файла).

5) **Иконка и бренд** (опционально)
   - Крупный план иконки на панели расширений.

Важно:
- Скриншоты должны отражать реальный интерфейс.
- Не используйте чужие бренды и материалы без разрешения.
