## Short description (up to 132 chars)
Autoscroll for any site and VK Music tools: export, mass add/remove tracks.

## Full description
SmartScroll & VK Music is a browser extension for quick page scrolling and convenient VK Music playlist tools.

Features:
- SmartScroll: auto scroll up/down, container selection, floating control panel.
- VK Music: mass add or remove tracks in playlist edit mode.
- Export tracks to TXT (format "Artist - Title").
- Single Stop button to halt all active operations.

How to use:
1) Open a page and launch the SmartScroll panel.
2) For VK Music, open playlist edit mode, scroll to the end, then run actions.
3) For export, scroll to the end and click "Export to TXT".

Notes:
- VK buttons work only in playlist edit mode.
- Large playlists may cause the browser to slow down.
